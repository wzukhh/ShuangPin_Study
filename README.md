# 双拼打字练习

一个现代化的双拼输入法练习工具，帮助用户快速掌握双拼输入技巧。

## ✨ 功能特性

- 🎯 **多种练习模式**
  - 单字练习：内置 3500 个中文常用字
  - 句子练习：内置 100+ 句子素材，涵盖所有键位组合，充分练习各种常用、生僻双拼按键组合
  - 可以自行在 `static/practice.js` 中，添加更多练习句子和练习单字

- ⌨️ **多双拼方案支持**
  - 内置多种双拼方案：`小鹤双拼`、`自然码双拼`、`微软双拼`、`搜狗双拼`、`智能ABC双拼`
  - 更多双拼方案可在 `static/keyboard-config.js` 中自行扩展

- 🎨 **现代化界面**
  - 响应式设计，适配各种屏幕尺寸
  - 明暗主题切换
  - 虚拟键盘可视化
  - 双拼码提示功能

- 📊 **实时统计**
  - 总字数统计
  - 完成进度追踪
  - 错误记录与分析
  - 用时统计
  - 完成后的详细报告

- 🎮 **交互体验**
  - 支持虚拟键盘和物理键盘输入
  - 实时输入反馈（正确/错误高亮）
  - 虚拟键盘按键实时高亮
  - 进度条可视化

## 🚀 快速开始

### 使用方式

1. 克隆或下载本项目
2. 直接在浏览器中打开 `index.html` 即可使用

无需安装任何依赖，纯前端实现，开箱即用！


## 📖 使用说明

1. **开始练习**
   - 点击"开始"按钮开始练习
   - 在输入框中输入当前高亮汉字对应的双拼按键（2个字母）

2. **输入方式**
   - 可以使用虚拟键盘点击输入（鼠标点击屏幕中的虚拟键盘）
   - 或使用物理键盘直接输入（需要切换为英文输入模式）

3. **输入反馈**
   - 输入正确后文本自动左移
   - 错误时字符显示红色，正确后显示绿色

4. **练习设置**
   - 可调整练习类型（单字/句子）
   - 可选择不同的双拼方案
   - 可开启/关闭双拼码提示

5. **查看统计**
   - 实时查看总字数、已完成、错误数和用时
   - 点击"查看错误记录"查看详细错误信息
   - 完成练习后会显示统计弹窗

## 🛠️ 技术栈

- **HTML5** - 页面结构
- **CSS3** - 样式和动画
- **JavaScript (ES6+)** - 核心功能实现
- **纯前端** - 无需后端，无需构建工具，无需安装任何编程环境

## 📁 项目结构

```
shuang-pin-pro/
├── index.html              # 主页面
├── static/
│   ├── style.css           # 样式文件
│   ├── script.js           # 主要逻辑
│   ├── practice.js         # 练习数据
│   ├── keyboard-config.js  # 键盘配置
│   └── pinyinPro.js        # 拼音处理
└── README.md               # 项目说明
```

## 🎯 核心功能模块

- **练习数据管理** (`practice.js`) - 单字和句子数据
- **键盘配置** (`keyboard-config.js`) - 双拼方案配置，可以添加自定义双拼方案
- **拼音处理** (`pinyinPro.js`) - 拼音转换和验证（使用开源pinyinPro）
- **主逻辑** (`script.js`) - 打字练习核心功能
- **样式系统** (`style.css`) - UI 样式和主题
